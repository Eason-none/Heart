# 患者端知识地图 — 信息架构文档 v0.1

**用途：** F1科普模块内容挂载索引 / 设计开发参考规格 / 未来 RAG 分块依据  
**目标用户：** 40–70岁确诊冠心病（含PCI/CABG术后）居家康复患者  
**语言原则：** 全部通俗化，禁用专业术语堆砌，以患者能理解的日常语言表达  
**版本日期：** 2026-06-01

---

## 节点层级约定

```
一级节点（Section）  →  8个主题
  └─ 二级节点（Subsection）  →  每个Section下4–5个子主题
       └─ 科普卡片（Card）  →  每个Subsection下2–4张卡片
```

**卡片ID命名规范：** `{section-abbrev}-{sub-abbrev}-{N}`  
例：`heart-anatomy-01`、`emergency-nitro-01`

**优先级说明：**
- `P0`：核心内容，MVP必须完成
- `P1`：重要内容，二期补充
- `P2`：扩展内容，按需添加
