## 一、首次评估问卷——需要采集的核心内容

文献高度一致地强调：评估是心脏康复的首要步骤，个性化评估是实施康复的前提和基础。

### 基本信息与病史
- 年龄、性别、身高、体重
- **腰围**（男≥90cm、女≥85cm为腹型肥胖，内脏脂肪重要代理指标）
- BMI（正常18.5~23.9，超重24~27.9，肥胖≥28）
- 当前诊断（稳定型心绞痛/PCI术后/CABG术后/急性心梗恢复期）
- 确诊或手术距今时间
- 既往史（高血压、糖尿病、脑卒中、肾病、睡眠呼吸暂停等）
- 目前用药情况（**重点：是否服用β受体阻滞剂**——直接影响心率计算方法）

### 运动能力基线评估工具（可内嵌于产品问卷）

**VSAQ问卷（退伍军人特定活动问卷）**  
13个运动项目，让用户选择能完成的最大项目，直接对应METs值。  
最大代谢当量 = 4.7 + 0.97×VSAQ分数 - 0.06×年龄  

| 序号 | 活动 | 对应METs |
|------|------|----------|
| 1 | 进餐、穿衣、伏案工作 | 1 |
| 2 | 洗澡、购物、步行8步 | 2 |
| 3 | 平地步行1~2个街区、携带杂货 | 3 |
| 4 | 庭院除草、打扫、轻木工 | 4 |
| 5 | 快速步行、交谊舞、洗车 | 5 |
| 6 | 打高尔夫（携带用具）、重型木工 | 6 |
| 7 | 步行爬山、重体力工作、搬25kg物体 | 7 |
| 8 | 搬重家具、快速爬楼梯、慢跑 | 8 |
| 9 | 中速骑车、锯木头、慢速跳绳 | 9 |
| 10 | 快速游泳、跑步9.5km/h | 10 |
| 11 | 负重爬楼梯2层、快速骑车 | 11 |
| 12 | 持续快跑1km/5min | 12 |
| 13 | 任何竞技赛动 | 13 |

**DASI问卷（Duke活动状态指数）**  
12项日常活动，回答"是/否"累计分数，推算峰值摄氧量：  
peakVO₂(ml/kg) = 0.43×DASI分数 + 9.6

### 高危识别触发器（任意一项阳性→高危降级）
- 静息时有胸痛/心绞痛症状
- 近期晕厥史
- 充血性心力衰竭（NYHA Ⅲ~Ⅳ级）
- 左室射血分数<40%（需用户填写超声心动图结果）
- 静息时复杂室性心律失常（有心律失常史）

### 精神心理基线
- **PHQ-9**（患者健康问卷9项）：抑郁筛查
- **GAD-7**（广泛焦虑问卷7项）：焦虑筛查
- **PSQI**（匹兹堡睡眠质量问卷）：睡眠评估

### 慢病合并情况
- 高血压（是/否，当前血压是否达标）
- 糖尿病（是/否，空腹血糖/糖化血红蛋白）
- 高血脂（是/否，LDL-C值）
- 是否吸烟（吸烟年限、每日支数）

## 二、危险分层标准

三级分层：低危/中危/高危  
**判断规则：每项都满足才是低危；任意一项满足高危标准即为高危**

| 指标 | 低危 | 中危 | 高危 |
|------|------|------|------|
| 左室射血分数 | ≥50% | 40%~50% | <40% |
| 功能储备（METs） | ≥7 | 5~7 | ≤5 |
| 运动时心绞痛 | 无 | 可有（≥5METs时） | 有（<5METs时） |
| 心电图缺血改变 | 无 | ST段下移<2mm | ST段下移≥2mm |
| 复杂室性心律失常 | 无 | 无 | 有 |
| 血流动力学反应 | 正常 | 正常 | 异常（心率变时不良或收缩压下降） |
| 心力衰竭 | 无 | 无 | 有 |
| 心理障碍（焦虑抑郁） | 无 | 无 | 有 |
| 猝死史 | 无 | 无 | 有 |
| 肌钙蛋白 | 正常 | 正常 | 升高 |

## 三、运动处方参数（Ⅲ期居家康复）

### 有氧运动处方

**靶心率计算——Karvonen法（心率储备法）**  
靶心率 = （最大心率 - 静息心率）× 强度% + 静息心率

- 最大心率：优先使用用户导入数据中的实测峰值心率；无数据时用 220-年龄 估算
- 静息心率：用户导入数据中的静息心率均值；无数据时用户填报
- 起始强度：40%~60%心率储备（低危/中危居家患者）
- 目标强度：逐步达到70%~90%心率储备

**重要：β受体阻滞剂用药切换**  
β受体阻滞剂会抑制心率反应，导致Karvonen法失效。用药患者应改用RPE评分法：
- RPE 11分：感觉轻松
- RPE 12~13分：感觉有点费力，还能正常说话，微微出汗（**目标区间**）
- RPE 15分：感觉吃力

**AT法（无氧阈法）**  
无氧阈前1分钟的心率或功率作为运动强度，是冠心病患者最佳推荐强度。

**运动参数总结（Ⅲ期居家康复）**

| 参数 | 建议值 |
|------|--------|
| 频率 | 每周3~5次 |
| 时长 | 20~40分钟（从15分钟起步，按周递增5分钟） |
| 热身 | 5~10分钟低强度有氧 |
| 放松 | 5~10分钟拉伸 |
| 总计 | 每周150分钟中等强度有氧 |
| 推荐形式 | 步行（首选）、慢跑、骑车、游泳、太极拳、八段锦 |

**运动强度的双重表达（产品设计原则）**  
同时给出：
1. 数字：心率区间（如95~115次/分）
2. 感受：对应RPE分级描述（"感觉有点费力，还能说话，微微出汗"）

### 抗阻运动处方

| 参数 | 建议值 |
|------|--------|
| 开始时机 | PCI术后3周，心梗后5周 |
| 频率 | 每周2~3次，隔天进行 |
| 起始强度 | 低强度20%~30% 1RM |
| 目标强度 | 逐步达到50%~70% 1RM |
| 注意事项 | 用力时呼气，放松时吸气，**严禁屏气（Valsalva动作）** |
| 上肢训练禁忌 | CABG术后3个月内不进行中到高强度上肢力量训练 |

**1RM估算方法（"理论最大负荷"法）**  
实测某重量的最大可重复次数，查表换算1RM：
- 重复10次 → 实测重量×1.33 = 理论1RM
- 训练重量 = 理论1RM × 目标强度%

### 柔韧与平衡训练

| 参数 | 建议值 |
|------|--------|
| 频率 | 每日 |
| 每个动作 | 维持15秒，重复≥4次 |
| 总时长 | 5~10分钟 |
| 形式 | 肌肉关节拉伸、太极拳、八段锦 |

### 合并症患者运动调整

**合并糖尿病：**
- 运动强度40%~60%心率储备（RPE 11~13分）
- 每周3~7天，至少150分钟中等强度
- 两次运动间隔不超过48小时
- **运动前血糖<5.6mmol/L不建议运动**
- 运动中备用糖果
- 避免在胰岛素作用高峰期运动（餐后1~2小时运动为宜）

**合并高血压：**
- 每天30~60分钟持续性或间歇性有氧运动
- 每周2~3次抗阻运动，40%~60%心率储备
- 运动中监测血压，收缩压>180mmHg时停止运动
- 未控制的高血压不建议抗阻训练

**合并肥胖：**
- 早期目标：每次至少消耗250~300kcal
- 目标：每周消耗1000~2000kcal（含日常体力活动）

## 四、运动后症状问卷——停止运动的指征

### 应立即停止运动并显示警示的症状
- 胸痛或胸闷
- 头晕或眼前发黑（黑矇）
- 严重气促（比平时明显）
- 心跳明显异常（过快/过慢/不规律）
- 出冷汗、恶心、面色苍白

### 量化停止指征
- 运动时心率增加>20次/分（与静息相比）
- 收缩压升高>40mmHg或下降>10mmHg
- 血氧饱和度<95%
- 心电图ST段变化（仅供参考，产品无法检测）

### 次日暂停/减量的信号
- 次日静息心率比基线高出10次/分以上（提示昨日运动量过大或身体状态变差）
- 运动结束30分钟后心率仍未恢复正常
- 次日出现明显疲劳感或气促

## 五、随访问卷内容框架

### 每周轻量随访（5分钟内完成）
1. 这周运动完成了几次
2. 是否出现任何新的不适症状
3. 睡眠质量如何（匹兹堡睡眠量表简化版）
4. 情绪状态（PHQ-9/GAD-7简化问题）
5. 是否有便秘或消化问题（**便秘→排便用力→血压骤升→心血管事件，有循证依据**）

### 每月完整随访（额外增加）
1. 血压记录（高血压患者：每日早晚测量，现在多少？）
2. 血糖记录（糖尿病患者：空腹血糖/餐后血糖）
3. 用药依从性（是否规律服药，有无自行停药）
4. 体重/腰围变化
5. 戒烟者有无复吸（**支架术后3个月是复吸高峰，需重点关注**）
6. 是否有新出现的症状或就医情况

### 文献推荐的随访频率
- 运动康复3个月内：每月1次完整随访
- 3个月后：每6个月1次
- 高危患者：出院后7天内首次随访
- 长期：1年后每年进行心血管综合评估

## 六、科普宣教——内容框架与推送逻辑

### 文献确认的13个核心健康教育主题

1. 心脏及冠状动脉基本结构与生理功能（疾病基础）
2. 冠心病发病机制、临床表现（诱发因素：重体力活动、情绪激动、暴饮暴食、寒冷刺激）
3. 心脏康复五大处方定义与益处（运动/药物/营养/心理/戒烟）
4. 危险因素识别与控制目标（高血压/高血脂/高血糖/吸烟/肥胖）
5. 主要不良心血管事件（MACE）症状识别与自救（硝酸甘油使用方法）
6. 如何在适当强度下安全运动（METs概念通俗化、RPE量表使用）
7. 心理调适：心梗后焦虑抑郁及应对方法
8. 地中海饮食原则与中国饮食本土化
9. 日常活动能量消耗指导（METs对照表）
10. 用药依从性：各类药物的作用与注意事项
11. 戒烟：益处、障碍（体重增加、戒断症状）与方法
12. 家庭急救：胸痛自救流程、何时拨打120
13. 回归社会：重返工作、驾驶、飞行、性生活的时机建议

### 术语通俗化三层结构（产品设计原则）
1. **替换**：用日常语言替代术语（静息心率→"安静坐着时心脏每分钟跳几次"；3MET→"相当于平地慢走的强度"）
2. **类比**：用熟悉事物解释陌生概念（冠状动脉狭窄→"就像水管内壁结了水垢，管子变细了，水流就小了"）
3. **按需展开**：默认显示通俗版，点击"了解更多"展示原始依据和文献来源

### 科普推送触发逻辑（循证行为改变）
- **行为节点触发**：完成首次运动后推"为什么要运动"；连续3天未运动推"坚持康复的意义"
- **数据异常触发**：静息心率连续升高推"睡眠与心脏健康"；饮食记录偏油腻推"地中海饮食入门"
- **时间节点触发**：每两周推送一篇危险因素管理内容；支架术后3个月推送戒烟相关科普
- **知识路径触发**：用户未主动访问科普模块满3天，公众号推送与当前康复阶段匹配的内容

### 院内vs院外教育重心对比（产品推送阶段参考）
| 阶段 | 推送重心 |
|------|---------|
| 注册初期（0~2周） | 什么是心脏康复、为什么要运动、危险信号识别（院内教育最佳时机） |
| 稳定康复期（1~6个月） | 危险因素管理、生活方式调整、运动进阶 |
| 长期维持期（6个月以上） | 为什么坚持很重要、心脏康复长期获益数据、回归社会指导 |

## 七、饮食指导——循证参数

### 地中海饮食核心要素（文献量化）
- 脂肪摄入占总能量≤30%，**饱和脂肪≤7%**
- 反式脂肪酸<总能量1%
- 每日胆固醇<300mg
- 每日食盐<6g（心血管患者建议<5g）
- omega-6多不饱和脂肪酸≤10%
- 植物甾醇每天2~3g
- 每日膳食纤维≥25g
- 完全戒酒最佳
- 每日用油<25g（约1.5汤匙）

### 合并症饮食调整

**合并糖尿病：**
- 主食定量，全谷物/杂粮豆类占1/3
- 血糖控制不好时减少水果摄入
- **进食顺序：先蔬菜→再肉→后主食**（降低血糖峰值）
- 避免含糖饮料、含酒精饮料

**合并高血脂：**
- 烹调油每天<30g
- 饱和脂肪供能比≤7%
- 每日胆固醇<300mg
- 增加膳食纤维

**合并肥胖：**
- 每天饮水1500ml以上
- 减重速度不宜过快，每周0.5~1.0kg为宜
- 三餐能量占比：早餐25%，午餐30%~35%，晚餐25%，加餐15%~20%

## 八、心理模块——筛查工具与处理逻辑

### 标准化评估工具

| 工具 | 用途 | 临界值 |
|------|------|--------|
| PHQ-9 | 抑郁筛查 | <5分正常；5~9轻度；10~14中度；≥15重度 |
| GAD-7 | 焦虑筛查 | <5分正常；5~9轻度；10~14中度；≥15重度 |
| PSQI | 睡眠质量 | >7分提示睡眠质量差 |

### 产品处理逻辑
- **轻度**（PHQ-9或GAD-7 5~9分）：AI对话支持、运动推荐（运动改善情绪有循证依据）、正念减压建议
- **中度**（≥10分）：强提示建议寻求专业心理援助，提供转介引导，产品不替代专业干预
- **重度**（≥15分）：建议精神专科就诊，产品只提供支持性科普

### 文献支持的重要结论
- **动机性访谈（MI）**是心脏康复健康教育中证据等级最高的干预形式之一（Level 1B，Grade A）
- 抑郁是冠心病患者运动危险分层的高危因素之一（与左室射血分数同等地位）
- 运动训练本身可改善焦虑抑郁状态，有循证依据

## 九、AI个性化反馈——数据识别与异常触发

### 心率数据导入后的计算逻辑

1. 从导入数据中提取：
   - 静息心率均值（最近7天清晨心率平均值）
   - 实测峰值心率（运动数据中的最高值）
2. 用Karvonen公式更新目标心率区间
3. 向用户展示更新前后的数值对比

### 数据异常触发规则

| 触发条件 | 推送内容 |
|---------|---------|
| 静息心率连续3天比基线高10次/分以上 | "你的静息心率近期有所升高，可能与睡眠、压力或身体状态有关，建议在下次随访时告知医生" |
| 运动心率长期无法达到目标区间下限 | "你的运动强度可能偏低，我们来调整一下方案" |
| 运动心率频繁超过目标区间上限 | "你的运动强度可能偏高，建议降低" |
| 连续3天未打卡运动 | 科普推送+依从性激励内容 |

### 重要语言边界
- 所有异常识别输出使用**观察性语言**，不使用诊断性语言
- 正确示例："你的静息心率近期有所升高"
- 错误示例："你的病情可能加重了"
- 所有异常均引导用户联系医生，不建议自行处理

## 十、用药模块——合规边界与科普内容

### 冠心病二级预防用药科普（产品可教育内容）

| 药物类型 | 用药目的 | 注意事项 |
|---------|---------|---------|
| 抗血小板药（阿司匹林）| 预防血栓 | PCI后需双联抗板12个月，不可自行停药 |
| β受体阻滞剂 | 控制心率、减少心肌耗氧 | 目标静息心率55~60次/分；会影响运动时心率反应 |
| ACEI/ARB | 保护心脏、降压 | 血压>110/60mmHg才适合使用 |
| 他汀类 | 降低LDL-C、稳定斑块 | 长期服用，定期复查肝功能 |

### 服药提醒的安全边界
- 提醒功能：✅ 提醒用药时间、频次
- 科普功能：✅ 解释药物用途、常见副作用
- 禁止功能：❌ 调整剂量、建议停药、推荐换药

## 十一、对产品设计的关键结论

### 1. β受体阻滞剂是被数字健康产品普遍忽略的重要变量
文献多处强调此药对心率的影响，产品必须在用药采集后触发逻辑切换。

### 2. RPE量表是最适合居家场景的强度监测工具
不依赖设备、与心率高度相关、用户学习成本低，应作为产品的核心强度表达方式。

### 3. 便秘是被忽略的心血管危险因素
机制：便秘→排便用力→血压骤升→心血管事件。随访问卷和科普内容均应覆盖。

### 4. 支架术后3个月是戒烟复吸高峰
产品随访模块应在这个时间点加强戒烟问询。

### 5. 院内早期是健康教育最佳时机
对应产品：用户注册初期（前2周）是信息接受度最高的窗口，应最大化科普推送密度。

### 6. 动机性访谈（MI）有最高级别循证依据
MI是Level 1B，Grade A的干预方法。产品的对话设计（包括FINDSELF的MI框架）有坚实的临床依据支撑。

### 7. 家属参与是提升依从性的重要手段
文献多处建议将家属纳入健康教育。公众号推送可作为触达家属的渠道（MVP阶段通过患者转发实现）。

### 8. 物理活动评估应作为"生命体征"常规采集
AACVPR声明明确指出PA应视为vital sign，建议产品每次随访均采集运动完成情况。
