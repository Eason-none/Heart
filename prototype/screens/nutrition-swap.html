这张表怎么用：每组替换都在演示营养页讲的三个方法。先看方法标签，理解"为什么这样换"，再试着把同一个方法用在表里没有的食物上。每次换掉一样，长期坚持效果明显。

→
替换为

午餐 / 晚餐场景
白米饭 → 混合主食，营养密度翻倍
🍚
白米饭
精制碳水，纤维少
→
混合
🍱
杂粮饭
糙米+红豆+小米
✓建议比例：白米 50% + 糙米/杂粮 50%，初次尝试可从 7:3 开始适应口感。膳食纤维增加约 3 倍。
⚡ 简单操作：买一袋「十谷米」或「杂粮包」，按 1:1 比例混入白米一起蒸，无需额外备料。

✓茶籽油是国内耐高温植物油中单不饱和脂肪酸含量最高的，适合中式大火爆炒。

✓不建议高温炒菜使用橄榄油（烟点低），凉拌 / 调汁场景最合适，保留多酚类抗氧化物质。

调味料替换（减盐关键） 方法③ ∨
日常炒菜 / 炖煮
减钠不减味的核心策略
🧂
普通食盐
100% 氯化钠
→
替换为
🫙
低钠盐 / 钾盐
钠减少 30%，含钾
✓低钠盐可在不改变咸度感知的前提下减少约 1/3 的钠摄入。注意：肾功能不全者使用前请咨询医生。
增香提鲜场景
不用味精 / 鸡精的天然增鲜方案
🫙
味精 / 鸡精
含大量谷氨酸钠（隐性盐）
→
替换为
🍄
香菇 / 虾皮 / 海带
天然谷氨酸，低钠增鲜
✓干香菇水泡发后的汤汁是天然增鲜利器，做汤和炖菜时加入效果显著，完全不需要味精。
⚡ 减盐小技巧：出锅前放盐（而非烹饪中途），相同量的盐味道感知更强，实际摄入更少。
