## 产品是什么

面向**确诊冠心病（含PCI/CABG术后）患者**的居家康复AI辅助管理工具。帮助低中危患者建立可持续的五大处方康复习惯，降低心血管事件风险与再住院率。

**形态：** 移动端优先Web App（不做独立App，避免医疗器械备案）。MVP不做主动推送通知。

---

## 用户边界

| 用户类型 | 处理方式 |
|---------|---------|
| 低危/中危冠心病患者 | 完整使用，中危额外强化随访 |
| 高危患者 | 只读模式：饮食/助手/科普/心理科普，运动Tab锁定 |
| 心肌病/瓣膜病/心衰 | 排除，不服务 |
| 健康成人 | 排除，不扩展 |

## 用户生命周期

```
注册
 └→ F3评估（5组≤30题，≤8分钟）
     └→ 评估摘要分叉
         ├→ 高危：只读模式，运动锁定
         └→ 低中危：功能引导（5张卡片）→ 开始康复
                                              │
                                    有氧3次/周 + 抗阻2-3次/周 + 柔韧
                                              │
                                        第12次运动 → 进入改善期
                                              │
                                        第18次运动 → 强化版里程碑
                                              │
                                        第90天 → 结局评估+康复小结 → 维持期
                                              │
                                    维持期：随访每2周，处方继续，自主管理
```

## F2 运动模块（最核心）

**每次运动流：** Check-in → 处方展示 → 执行（计时）→ 反馈 → AI总结卡片

**Check-in三层状态机：**

| 状态 | 触发条件 | 处理 |
|------|---------|------|
| 好 | 睡眠好/一般 + 精力好；或睡眠一般 + 精力好 | 正常处方 |
| 一般 | 睡眠一般AND精力比较累；或睡眠好AND精力比较累 | 降1RPE档，直接进处方 |
| 差 | 睡眠很差 OR 精力很累（任一触发） | 四选一：①有氧降强 ②抗阻Borg≤12 ③柔韧 ④PLDB呼吸 |
| 症状触发 | 胸痛/胸闷/心慌/头晕等任意勾选 | 锁定当日，就医建议 |

**强度计算：**
- 有手环：Karvonen法（HRR）+ RPE双轨
- 无手环：RPE主导，心率仅参考
- β阻滞剂用户：仅RPE，心率不参与任何触发逻辑

**RPE四档（名称统一，描述按类型区分）：**
- 有氧：Talk Test疲劳感（"能说话但不能唱歌"）
- 抗阻：RIR留力法（"还能再做3-5次" = 目标区间）

**三层强度调整触发：**
- 层1（单次）：记录，不触发
- 层2（14天内3次窗口）：RPE/恢复/HR信号触发升降
- 层3（每周随访综合）：静息HR趋势+6MWT趋势
- 差状态日/一般状态日不计入层2计数

## F3 首次评估

**5组≤30题：**
1. 基本信息（诊断类型/手术月数/身高体重）
2. 心脏功能+高危筛查（LVEF两步法/高危三问）
3. 慢病与用药（β阻滞剂/合并症/吸烟）
4. VSAQ功能储备（→METs→初始处方强度）
5. PHQ-2+GAD-2心理基线快筛

**高危触发：** 高危三问任意"是" OR PHQ-2+GAD-2任意≥3

**VSAQ是唯一功能储备量表，DASI已完全移除。**

## F8 随访

**每周5题轻量：** 运动次数/不适/睡眠/情绪/便秘（+慢病条件题）

**每月扩展：** 血压血糖/用药依从（M4）/体重腰围/戒烟/新发症状（M3）/VSAQ复测+椅子站立

**M3（新发症状）：** 危险症状核查清单 → 有危险症状则暂停有氧+抗阻直至就医复诊

**M4（漏服较多）：** 暂停有氧+抗阻 → 解锁条件：下次随访M4正常 OR 主动标记恢复用药（条件触发，不是天数触发）

**心理分级响应（PHQ-9）：**
- 5-9轻度：运动正常+心理科普卡频率上调
- 10-14中度：+建议与医生/咨询师沟通
- ≥15重度：暂停运动+明确建议就诊
