💡 主食选购原则
选全谷物、杂粮优先，精制米面血糖指数高，长期大量食用不利于体重和血糖管理。
🌾
燕麦（整粒 / 钢切）
β-葡聚糖降低 LDL 胆固醇，饱腹感强，GI 低
优选
🍚
糙米 / 杂粮饭
保留麸皮和胚芽，膳食纤维是白米的 3–4 倍
优选
🫘
红薯 / 山药
钾含量高，有助于维持血压平衡，可替代部分主食
优选
🥐
全麦面包（注意成分表）
需确认全麦粉排第一位，含添加糖的品牌需控量
适量
🍜
精制白米 / 白面条
精制碳水 GI 高，大量食用影响血糖，不利体重管理
减少
🍞
普通白面包 / 起酥包
高 GI + 含反式脂肪酸（起酥油），双重不利因素
避免
💡 蛋白质选购原则
优选鱼类和豆类；禽肉去皮；红肉每周不超过 500g；加工肉制品尽量少买。
🐟
三文鱼 / 鲈鱼 / 带鱼
富含 Omega-3 脂肪酸，可降低甘油三酯，保护心脏
每周≥2次
🫘
豆腐 / 毛豆 / 鹰嘴豆
植物蛋白质，饱和脂肪低，可替代部分动物蛋白
优选
🥚
鸡蛋（全蛋）
优质蛋白，适量摄入（每天 1 个），不必过度限制
每日 1 个
🍗
鸡肉 / 鸭肉（去皮）
去皮后饱和脂肪显著降低，是优质白肉来源
去皮食用
🥩
瘦牛肉 / 猪里脊
铁含量高，但饱和脂肪较多，每周控制在 500g 以内
适量
🌭
香肠 / 培根 / 腊肉
高钠、高饱和脂肪、含亚硝酸盐，心血管风险明确
避免
💡 蔬菜选购原则
每天 ≥ 400g 蔬菜，深色蔬菜（绿/橙/红）占一半以上。品种多样优于大量单一品种。
🥦
西蓝花 / 菠菜 / 空心菜
富含叶酸、维生素 K、抗氧化素；注意服华法林者控量
深色优先
🍅
番茄 / 胡萝卜 / 南瓜
番茄红素和 β-胡萝卜素，抗氧化，加热后吸收更佳
优选
🧄
大蒜 / 洋葱
含有机硫化物，研究显示有助于轻度降低胆固醇和血压
调味优选
🥔
土豆 / 芋头
淀粉含量高，建议作为主食替代计算，非额外添加
计入主食量
🥫
腌制蔬菜（泡菜/咸菜）
钠含量极高，100g 泡菜含盐量可超出全天推荐量
极少食用
💡 油脂选购原则
选不饱和脂肪酸为主的植物油；控制总用量（每天 25–30g）；坚果是优质脂肪来源。
🫒
橄榄油（初榨）
单不饱和脂肪酸最丰富，抗氧化，地中海饮食核心
优选
🌰
核桃 / 杏仁 / 亚麻籽
含 Omega-3、维生素 E；每天一小把（约 30g）为宜
每日一把
🌻
菜籽油 / 茶籽油
国内易得，Omega-3 / Omega-6 比例较优，耐高温
中餐适用
🧴
花生油 / 玉米油
Omega-6 偏高，少量使用问题不大，注意总量
控量使用
🧈
猪油 / 黄油 / 椰子油
饱和脂肪酸高，过量摄入升高 LDL 胆固醇
减少
🍟
起酥油 / 氢化植物油
含反式脂肪酸，明确增加心血管事件风险，查成分表
严格避免
💡 乳制品选购原则
低脂或脱脂优先；原味无添加；每天 250–500ml 奶或等量乳制品。
🥛
低脂纯牛奶 / 脱脂奶
钙质丰富，低脂版本饱和脂肪少，性价比高
每日 250ml
🍶
无糖原味酸奶
益生菌有助于肠道健康，注意查成分表确认无添加糖
原味无糖
🧀
奶酪（低钠品种）
蛋白质和钙丰富，但钠含量差异大，选低钠品种
查钠含量
🧋
含糖奶茶 / 调味奶
糖分和热量极高，对血糖和体重管理均不利
避免
💡 零食 & 饮料选购原则
尽量选天然食物；看营养成分表，关注钠、糖和脂肪含量；含糖饮料是最容易被忽视的心血管危险因素之一。
🍎
新鲜水果（低糖品种）
蓝莓、苹果、橙子富含黄酮和膳食纤维，优先整果
整果优先
🫐
无盐混合坚果
健康脂肪来源，每天 30g，选无盐无糖品种
无盐无糖
🍫
黑巧克力（可可 70%+）
含黄烷醇有护心作用，但热量较高，每次 1–2 片
少量
🥤
含糖碳酸饮料 / 果汁饮料
每瓶可含 8–12 茶匙糖，液态糖比固态更快升高血糖
避免
🍪
饼干 / 薯片 / 膨化食品
高钠、高饱和脂肪，部分含反式脂肪，营养密度极低
