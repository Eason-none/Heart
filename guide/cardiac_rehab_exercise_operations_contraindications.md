# 冠心病心脏康复运动项目操作说明与禁忌症手册

> **适用人群：** 冠心病（CHD）、PCI/CABG 术后康复期患者  
> **文件用途：** 临床指导 + 患者教育参考  
> **依据来源：** AHA《心血管疾病抗阻训练》（Circulation 2000）· ACSM 2022 · AHA/AACVPR 2024 · 北欧健走 RCT（Canadian J Cardiology 2022）· 游泳/水中运动系统综述（2015/2022）· 太极荟萃分析（RCM 2024）· 八段锦系统综述（2022–2024）· ACPICR 游泳指南（2023）

## 通用禁忌症（所有运动项目均适用）

以下情况不论何种运动，**均须暂停，待临床稳定后重新评估**：

| 情况 | 说明 |
|------|------|
| 不稳定型心绞痛 | 近期静息或轻微活动时出现胸痛 |
| 急性失代偿心衰 | 明显气短、水肿加重、端坐呼吸 |
| 复杂室性心律失常 | 反复室速、室颤风险高 |
| 未控制的高血压 | 收缩压 ≥ 160 mmHg 和/或舒张压 ≥ 100 mmHg |
| 血流动力学不稳定的严重瓣膜病 | 重度主动脉瓣狭窄、严重二尖瓣反流 |
| 严重肺动脉高压 | 右室收缩压 > 60 mmHg |
| 腔内血栓 / 近期血栓性静脉炎 | 含或不含肺栓塞 |
| 急性全身性炎症或感染 | 发热、感染活动期 |
| 妨碍安全运动的肌肉骨骼问题 | 无法维持安全体位或动作 |

> **来源：** Piepoli et al. Eur Heart J 2024；PMC10989450（心衰 CR 综述，2024）

## 一、北欧式健走（Nordic Walking）

### 1.1 什么是北欧式健走？

起源于芬兰的越野滑雪夏季训练，使用特制手杖配合自然步行节奏，全身 80–90% 的肌群参与（普通步行仅 50–70%）。

**对比普通步行的优势：**
- 能耗增加 18–67%
- 功能容量（6MWT）改善显著优于 HIIT 和 MICT（RCT 2022）
- 分散腿部关节压力，对膝髋关节友好
- 辅助平衡，减少跌倒风险

### 1.2 手杖选择与准备

**手杖高度：** 手握杖顶时，前臂与地面呈 **90°**（即肘部成直角）。

**杖尖选择：**
- 橡胶杖头 → 适合城市铺装路面（人行道、公园小道）
- 金属尖头 → 适合自然土路、山径

**握持系统：** 北欧健走专用手杖配有手套式固定带（非普通登山杖的松弛腕带），用掌心而非手指发力推杖。

**着装建议：** 穿有支撑性的运动鞋，宽松透气衣物。

### 1.3 动作要领（分步学习）

**第一步：热身（普通步行 3–5 分钟）**

先不持杖，以正常步速走热身，让肌肉和心率逐步升温。

**第二步：基础技术——对侧摆臂**

> 北欧健走的核心节奏是：**右脚迈出，左臂推杖；左脚迈出，右臂推杖**（与普通走路摆臂完全相同的对侧协调）

1. 站立，双手自然握持手杖，杖身斜后方拖行
2. 右脚向前迈步，同时**左臂随动向前**，将左侧手杖杖尖植于右脚前方约 30–45 cm 处
3. 以手掌向后推压手杖，感受核心和背部肌肉同时用力
4. 左脚跟上，右臂前摆，右侧手杖重复上述动作
5. 步幅略大于普通步行，保持抬头、收腹、肩膀放松

**第三步：进阶——双杖同步推（Double Poling）**

适合体能较好、已熟悉单侧技术的患者：
1. 双杖同时植于身体前方，对称放置
2. 双臂同步向后推，走 3 步后重复
3. 适合坡路缓降或平地强度提升

### 1.4 推荐参数（来自心脏康复 RCT）

| 参数 | 推荐值 |
|------|--------|
| 频率 | 每周 **2–3 次** |
| 每次时长 | **30–60 分钟**（含热身/冷身） |
| 强度 | RPE 12–14（可正常说话） |
| 进阶方式 | 先增加时长，再增加频率，最后提升速度或坡度 |
| 初学者过渡方案 | 前 2 周每次仅 25% 时间持杖，其余时间普通步行 |

> **来源：** Terada et al. Canadian Journal of Cardiology 2022 RCT；Harvard Health 2024 指导文章

### 1.5 特有禁忌症与注意事项

| 情况 | 处理 |
|------|------|
| CABG 术后 **< 8–10 周** | 胸骨未愈合，**禁止上肢推杖发力**；待外科许可后方可开始 |
| 严重上肢关节炎或肩袖损伤 | 避免，或仅做不用力的杖辅助平衡步行 |
| 步态严重不稳 / 平衡功能极差 | 须先进行平衡训练，或在监督下从普通步行开始 |
| 在崎岖地面或雨后湿滑路面 | 使用防滑金属尖头，降低速度 |

## 二、抗阻训练（哑铃 / 弹力带 / 器械）

### 2.1 开始前提条件

以下条件须**同时满足**方可开始抗阻训练：

- LVEF > 40%（中—良好的左心室功能）
- 运动耐量 > 5–6 METs（无心绞痛症状或缺血性 ST 改变）
- 血压控制稳定（< 160/100 mmHg）
- 无不稳定心绞痛
- CABG 患者：胸骨完全愈合（术后 **≥ 8–10 周**，需外科确认）

> **来源：** AHA《心血管疾病患者抗阻运动》Circulation 2000；ACSM 2022

### 2.2 操作要领

**呼吸原则（最关键）：**
> **用力时呼气，放松时吸气。绝对禁止屏气（Valsalva 动作）。**
> 屏气会瞬间升高胸腔内压和动脉血压，对心脏患者极度危险。

**动作节奏：**
- 每个动作 **慢速控制**，上举 2 秒，下放 3–4 秒
- 不要借助惯性和甩动

**具体动作举例（适合居家）：**

**上肢——哑铃肱二头肌弯举**
1. 坐位，背部挺直，双手各持哑铃，掌心向上
2. 呼气，缓慢弯肘将哑铃举至肩部
3. 吸气，缓慢还原
4. 保持肘部贴近躯干，肩膀不耸动

**下肢——坐姿腿部抬伸**
1. 坐于椅边，背部挺直
2. 呼气，缓慢伸直膝关节抬腿至水平
3. 吸气，缓慢还原
4. 适合无哑铃时用自身重量练习

**核心——弹力带划船**
1. 将弹力带固定于稳固物体（门把手高度）
2. 坐位，双手握带，肘部向后拉，肩胛夹紧
3. 用力时呼气，还原时吸气
4. 适合代替器械训练的居家方案

### 2.3 推荐参数

| 参数 | 初始阶段 | 进阶阶段 |
|------|---------|---------|
| 负荷 | 40–50% 1RM（可完成 12–15 次不感费力） | 60–70% 1RM（上肢）/ 70–80% 1RM（下肢） |
| 组数 | **1 组**起始 | 逐步进阶至 **3 组** |
| 重复次数 | **10–15 次/组** | 同左 |
| 组间休息 | **1–2 分钟** | 同左 |
| RPE 目标 | 11–13（轻松–有点费力） | 同左 |
| 频率 | 每周 **2 次**（非连续日） | 进阶至 **3 次/周** |
| 负荷进阶信号 | 能舒适完成 15 次上限 → 增加约 **5%** 负荷 | 同左 |

> **来源：** AHA/AACVPR 2024 科学声明；EAPC 立场声明（EJPC 2022）；Fan et al. Front Cardiovasc Med 2021 系统综述

### 2.4 抗阻训练禁忌症

| 绝对禁忌 | 说明 |
|---------|------|
| 不稳定型心绞痛 | 静息或轻微用力诱发 |
| 未控制高血压（≥ 160/100 mmHg） | 须先药物控制 |
| 未控制的心律失常 | 室速/室颤风险 |
| 急性或失代偿心衰 | 未评估和有效治疗者 |
| 严重瓣膜病 | 重度主动脉瓣狭窄或关闭不全 |
| 肥厚型梗阻性心肌病 | 高强度用力加重流出道梗阻 |
| CABG 术后 < 8–10 周 | 胸骨骨愈合前禁止上肢抗阻 |

> **来源：** AHA Circulation 2000（Contraindications to resistance training）；ClinicalTrials NCT02674659

## 三、游泳 / 水中运动

### 3.1 水中运动对心脏的特殊影响

水中静水压使外周血液回流增加，心脏前负荷升高，心脏需要比同等陆地运动**做更多工**。这对稳定期患者是有益刺激，但对不稳定或失代偿患者是风险。

水温也至关重要：
- 推荐水温：**33–34°C**（温水池）
- 过冷（< 26°C）：引起血管收缩、血压骤升，心脏负荷剧增
- 过热（> 38°C）：血管扩张、血压下降，有晕厥风险

### 3.2 操作要领

**入水前准备：**
1. 陆上热身 5–10 分钟（拉伸、步行）
2. 逐步入水，先将手脚浸入，再缓慢进入腰部深度
3. 不要跳入水中（突然冷刺激可诱发心律失常）

**主运动阶段（20–30 分钟）：**
- 游泳：蛙泳、自由泳均可，控制中等速度（能正常呼吸说话）
- 水中步行（水中跑）：在胸腰部深度水中步行，利用水的阻力
- 水中体操：站立于池边，进行上下肢协调动作

**冷身（5–10 分钟）：**
- 减慢游速或改为水中慢走
- 出水前先在浅水区站立休息 2–3 分钟，避免直立性低血压

**出水注意：**
- 缓慢出水，扶扶手或台阶
- 立即擦干，及时保暖（尤其秋冬季）

### 3.3 推荐参数

| 参数 | 推荐值 |
|------|--------|
| 水温 | **33–34°C** |
| 每次时长 | 主运动 **20–30 分钟** |
| 强度 | RPE 4–6（0–10 量表）/ 中等强度 |
| 频率 | 每周 **3 次** |
| 适用阶段 | Phase II–III（稳定期） |

> **来源：** 水中运动 CAD 系统综述（ResearchGate 2022）；稳定心衰水中运动系统综述（ScienceDirect 2015）；ACPICR 游泳指南（2023）

### 3.4 游泳/水中运动禁忌症

| 禁忌情况 | 说明 |
|---------|------|
| 心梗 / 支架术后 **< 4–6 周** | 伤口未愈合，须待稳定期 |
| CABG 术后 **< 12 周** | 胸骨及手术切口未完全愈合 |
| 起搏器 / ICD 植入后 **< 6 周** | 皮肤切口未闭合，感染风险 |
| 急性心衰（失代偿期） | 前负荷进一步增加危险 |
| 开放伤口 / 皮肤感染 | 感染风险，同时污染水质 |
| 未控制的高血压 | 水压加重心脏负担 |
| 严重直立性低血压 | 出水后易晕厥 |
| 传染性皮肤病或发热 | 公共安全考量 |
| 不会游泳或极度恐水 | 选择水中步行替代 |

> **来源：** ACPICR 游泳与心脏健康指南；MyMovementMedicine 心脏患者游泳指南（2025）；AHA 瓣膜/心衰禁忌

## 四、太极拳（Tai Chi）

### 4.1 适合心脏康复的太极流派

推荐优先选择以下两种（动作较简化，适合老年和体弱患者）：

- **杨式 24 式简化太极拳**：国家体育总局推广版本，动作平稳，强度约 3–4 METs
- **太极（坐式改良版）**：供不能站立练习的患者使用，已有 RCT 证实中风后有效

**不建议**高难度的陈式太极（跳跃、发力动作多）或竞技套路。

### 4.2 基本操作要领

**站立准备姿势：**
1. 两脚与肩同宽，脚尖略向外分开约 15°
2. 膝关节微屈（不要完全伸直），重心居中
3. 脊柱自然直立，收下颌，目视前方
4. 肩膀放松下沉，双手自然垂于体侧

**运动中核心要求：**
- **动作缓慢匀速**，全程保持"太极劲"——绵柔、连贯、不间断
- **呼吸配合动作**：开势（手展开、下沉）时吸气；合势（手收拢、推出）时呼气
- **全程不屏气**，保持自然呼吸节奏
- **重心转换要稳**：单腿支撑时确认平衡，必要时扶椅或墙

**第一节示范——起势（适合入门）：**
1. 站立，双臂缓慢向前平举至肩高，掌心向下（吸气）
2. 屈膝微蹲，双臂缓慢下按至腹前（呼气）
3. 还原站立，重复 4–8 次

**注意：** 初学者建议观看教学视频（国家体育总局官方太极推广视频）配合练习，而不是仅靠文字指导。

### 4.3 推荐参数

| 参数 | 推荐值 |
|------|--------|
| 频率 | 每周 **3–5 次** |
| 每次时长 | **30–60 分钟** |
| 强度 | RPE 10–13，约 3–4 METs |
| 疗程 | ≥ **12 周**（RCT 通常为 12–24 周） |
| 适用阶段 | Phase II–III；状态不良调整日 |

> **来源：** Fan et al. Reviews in Cardiovascular Medicine 2024 荟萃分析（N=2095）；Chen et al. 2020 系统综述

### 4.4 太极禁忌症

| 禁忌情况 | 说明 |
|---------|------|
| 急性关节炎发作 | 关节痛急性期避免下肢承重动作 |
| 严重平衡障碍 | 坐式改良太极可替代 |
| 严重骨质疏松 / 近期骨折 | 转体动作有跌倒骨折风险，须医生评估 |
| 通用 CR 禁忌症（见表头） | 同适用 |

**特别注意：** 在户外练太极时，避免极寒或极热天气，寒冷会使血管收缩升高血压。

## 五、八段锦（Baduanjin Qigong）

### 5.1 为什么推荐八段锦？

- 动作固定、标准统一（国家体育总局 2003 年规范版本）
- 无需设备，室内室外均可
- 强度约 **3–4 METs**（中等偏低），适合中危患者
- 已有系统综述证实：改善 CHD 患者心肺功能、LVEF、6MWT 和生活质量

> **来源：** Wang & Xu 荟萃分析（Evidence-Based Complementary Medicine 2022）；Baduanjin 心脏术后系统综述（Medicine 2024）

### 5.2 八节动作简介与呼吸配合

以下为国家标准版本的八节动作要点，每节 8 个呼吸循环：

| 节次 | 动作名称 | 动作要点 | 呼吸配合 |
|------|---------|---------|---------|
| 预备 | 调息起势 | 站立，两手交叠置于丹田（小腹），调整呼吸 3 次 | 腹式深呼吸 |
| 第一节 | 两手托天理三焦 | 双手指交叉，翻掌向上托举过头顶，缓慢还原 | 托举时吸气，还原时呼气 |
| 第二节 | 左右开弓似射雕 | 两脚开立，马步，一手推弓，一手拉弦，交替左右 | 开弓时呼气，还原时吸气 |
| 第三节 | 调理脾胃须单举 | 一手上托，一手下按，交替左右 | 上托时吸气，还原时呼气 |
| 第四节 | 五劳七伤往后瞧 | 两脚站立，头缓慢转向后方，目视后方，交替左右 | 转头时呼气，还原时吸气 |
| 第五节 | 摇头摆尾去心火 | 马步下蹲，躯干缓慢左右摇摆 | 摇摆时缓呼，还原时吸气 |
| 第六节 | 两手攀足固肾腰 | 两手沿体后滑下至踝部，前屈，再还原 | 前屈时呼气，还原时吸气 |
| 第七节 | 攒拳怒目增气力 | 马步，握拳缓慢推出，目视前方，交替左右 | 推拳时呼气，还原时吸气 |
| 第八节 | 背后七颠百病消 | 两脚踮起，缓慢落踵 7 次，每次感受振动 | 踮起时吸气，落踵时呼气 |

**注意：** 心脏患者练习时，以下节次需特别注意：
- **第一节（双手托天）**：CABG 术后 < 10 周患者，上举高度不超过肩部
- **第六节（攀足）**：有腰椎病或直立性低血压者，前屈幅度减小，起身务必缓慢
- **第八节（踮脚落踵）**：高龄或骨质疏松者降低落踵力度，或跳过此节

### 5.3 推荐参数

| 参数 | 推荐值 |
|------|--------|
| 频率 | 每周 **3–5 次** |
| 每次时长 | **20–40 分钟**（含调息） |
| 强度 | RPE 10–13，约 3–4 METs |
| 疗程 | ≥ **12 周** |

### 5.4 八段锦禁忌症

| 禁忌情况 | 处理建议 |
|---------|---------|
| CABG 术后 < 10 周（上举动作） | 跳过第一节或限制上举高度 |
| 严重腰椎病（前屈动作） | 跳过第六节或减小幅度 |
| 严重骨质疏松（振踵动作） | 跳过第八节或轻踮不落踵 |
| 平衡功能严重受损 | 全程扶椅或坐姿改良练习 |
| 通用 CR 禁忌症 | 同见表头 |

## 六、瑜伽（Yoga）

### 6.1 适合心脏康复的瑜伽形式

推荐选择：
- **哈他瑜伽（Hatha Yoga）**：动作缓慢，侧重呼吸与拉伸，心血管负荷最低
- **阴瑜伽**：被动保持姿势，放松为主，适合调整日

**不推荐**的高强度形式：热瑜伽（Bikram，38–42°C 高温，对心脏患者风险极高）、流瑜伽（Vinyasa，节奏快速）。

### 6.2 基本操作要领

**呼吸原则：** 所有姿势均配合缓慢腹式呼吸，**绝对禁止屏气**。

**几个适合心脏患者的基础姿势：**

**山式（站立呼吸）**
1. 双脚并拢站立，脊柱延展
2. 吸气时手臂缓慢侧举至头顶，呼气时缓慢放下
3. 重复 5–8 次

**猫牛式（核心、腰背灵活性）**
1. 四肢撑地，双手在肩下，膝在髋下
2. 吸气时腰部下沉、头部上扬（牛式）
3. 呼气时腰部拱起、头部低垂（猫式）

**仰卧扭转（放松）**
1. 仰卧，屈膝，双膝缓慢向一侧倒
2. 双臂展开，感受腰脊旋转，保持 30–60 秒
3. 吸气还原，换侧重复

### 6.3 推荐参数

| 参数 | 推荐值 |
|------|--------|
| 频率 | 每周 **2–3 次** |
| 每次时长 | **30–60 分钟** |
| 强度 | RPE 10–12（轻松） |
| 适用场景 | Phase II–III；主观状态不良的调整日 |

> **来源：** Li et al. VHRM 2021（6 项 RCT 综述）；系统综述（ScienceDirect 2020，7 项 RCT，4671 人）

### 6.4 瑜伽禁忌症

| 禁忌情况 | 说明 |
|---------|------|
| **热瑜伽（Bikram）** | 高温引起外周血管扩张，血压急降，心脏患者绝对禁止 |
| 严重骨质疏松 | 深度前屈和扭转有脊椎压缩性骨折风险 |
| 青光眼 | 倒立体位增加眼内压 |
| 近期腹部手术 | 核心发力动作禁止 |
| CABG 术后 < 8 周 | 上肢支撑体位（平板支撑、下犬式）禁止 |
